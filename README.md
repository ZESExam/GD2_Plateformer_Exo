# GD2_Plateformer_Exo
Mini prototype de jeu de plateforme en blocking volume 
Concept : Mini jeu pour tester des mécaniques simple pour ensuite faire un plateformer dans le noir.
Le jeu contient quelques mecaniques simple :
—Récuperer differentes pieces pour augmenter son score
—Récuperer une clé dans un level pour pouvoir aller débloquer un passage dans un autre level
—Interagir avec des elements pour pouvoir jouer 


Deuxième partie du devoir:
Rajout de l'enigme:
Aller dans le niveau 3 pour activer les 3 boutons qui sont dissimuler dans le parcours puis retourner dans le niveau 1.
Ce qui va avoir pour effet d'allumer les 3 lumières de la porte du niveau 4 en vert.
Une fois les 3 lumières allumer en vert la porte s'ouvre et on a accès au dernier niveau de fin.
